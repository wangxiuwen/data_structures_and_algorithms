# 169. 多数元素

给定一个大小为 n 的数组，找到其中的多数元素。多数元素是指在数组中出现次数大于 ⌊ n/2 ⌋ 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。

示例 1:

```
输入: [3,2,3]
输出: 3
```

示例 2:

```
输入: [2,2,1,1,1,2,2]
输出: 2
```

解法一

哈希表

```python
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
      d = {}
      n = len(nums)
      for i in range(len(nums)):
        if nums[i] not in d:
          d[nums[i]] = 1
        else:
          d[nums[i]] += 1

        if d[nums[i]] > n/2:
          return nums[i]
```
