# 102. 二叉树的层次遍历

给定一个二叉树，返回其按层次遍历的节点值。 （即逐层地，从左到右访问所有节点）。

例如:

```
给定二叉树: [3,9,20,null,null,15,7],

    3
   / \
  9  20
    /  \
   15   7
```

返回其层次遍历结果：

```
[
  [3],
  [9,20],
  [15,7]
]
```

## 解法一

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def levelOrder(self, root: TreeNode) -> List[List[int]]:
      if not root:
        return []

      result = []

      level = 0
      queue = []
      queue.append(root)
      while queue:
        result.append([])
        for i in range(len(queue)):
          node = queue.pop(0)
          result[level].append(node.val)

          if node.left:
            queue.append(node.left)
          if node.right:
            queue.append(node.right)
        
        level += 1
      return result
```
